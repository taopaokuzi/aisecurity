## 任务目标

实现员工侧最小可用前端，包括申请页、申请状态页和申请详情页。

## 范围

- 申请表单页
- 申请提交成功页
- 申请列表或状态页
- 申请详情页
- 评估结果与状态展示

## 交付物

- Next.js 员工侧页面
- 与申请/评估接口的联调
- 基础前端测试

## 完成定义

- 员工能提交申请并查看当前状态
- 能展示风险等级、建议权限和审批状态
- 页面可在桌面端和移动端基本可用
